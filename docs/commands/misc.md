# Miscellaneous Commands

Small commands that do not need their own full guide.

## `stado version`

Prints the resolved stado version. Dev builds may report
`0.0.0-dev` or the module version from Go build metadata.

```sh
stado version
```

## `stado config-path`

Loads config and prints the config file path stado uses.

```sh
stado config-path
```

Because `config.Load` also ensures the config directory and default
system prompt template exist, this command can create
`~/.config/stado/system-prompt.md` on first run.

## `stado completion`

Generated by Cobra. Emits shell completion scripts.

```sh
stado completion bash
stado completion zsh
stado completion fish
stado completion powershell
```

Use `stado completion <shell> --help` for shell-specific installation
notes.
